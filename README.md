# Spam Email Classifier

A beginner ML project that classifies emails as Spam or Not Spam using Naive Bayes algorithm in Python.

## Technologies Used
- Python
- scikit-learn
- pandas

## How to Run
1. Install libraries: pip install scikit-learn pandas
2. Run: python spam_classifier.py
3. Enter any email message when asked
